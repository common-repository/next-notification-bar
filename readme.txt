== Description ==

Next Notification Bar is a WordPress Notification Bar plugin that allows you to create notification bars to notify your visitors.

[youtube https://www.youtube.com/watch?v=KPl1C4mO9jI]

= Features =

- Modern and clean UI
- Live preview when you customize the notification bar
- Preview your notification bar on front-end
- HTML content
- Choose various themes, positions, alignments
- Show/Hide button
- Options to show or hide: All pages / Homepage / Only posts / Only pages
- Show/Hide Close button
- Set status for notification bar

== Installation ==

1. Go to the WordPress Dashboard > Plugins > Add New.
2. Search for "Next Notification Bar".
3. Install, then Activate it.
